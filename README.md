# <p align='center'>SoundHub</p>

#### A music streaming platform <b>Attempt</b> using laravel

### Get Started
 
 `git clone https://github.com/epikoder/soundhub`

`cd soundhub`

`composer install`

`cp .env.example .env`

`php artisan key:generate`

Create a database on Mysql and update the 
fields in .env

`DB_CONNECTION=mysql`

`DB_HOST=127.0.0.1`

`DB_PORT=3306`

`DB_DATABASE=name`

`DB_USERNAME=root`

`DB_PASSWORD=`

Start server

`php artisan serve`
